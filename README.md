# amavc - Android Markets App Version Check
***
<br/>

## 简介
这是一个用于快速检测指定app在目前国内流行的诸多安卓市场中的最新发布版本的工具，可以帮助跟踪android app的对外发布情况，进而保证指定app在所有安卓市场中的发布版本都是最新的。
